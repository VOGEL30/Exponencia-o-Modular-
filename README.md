\section*{Descrição do Programa}

Este programa implementa o algoritmo de exponenciação modular eficiente, também conhecido como exponenciação rápida. Ele calcula $a^k \mod n$ de forma otimizada, reduzindo o número de multiplicações necessárias. O programa é interativo e exibe, passo a passo, o processo de cálculo, tornando o funcionamento do algoritmo transparente e compreensível.
